We can first start by a simple triangle shape from which we've removed the bottom.
Then, by rotating around the center of the canvas, we can duplicate this shape at every 90 degrees, creating a symmetric shape.
Using this basis, we can then add lines between the two extremities.
We can repeat the process with a different shape.
With this shape, several lines align when rotating around the center point, creating different shapes in the process.
By adding lines within the basic shape, we can create overlaps and moire-like patterns in the shape.
We can repeat the same process with a slightly different shape in which the extremities go only halfway down the height.
By applying the rotation, the diamond created is slightly bigger, but the boundaries of the shape are similar.
When introducing more lines, the overlapping effect is even more intense.
Adding the initial red shape creates more intensity and fills the negative space inside the shape.
