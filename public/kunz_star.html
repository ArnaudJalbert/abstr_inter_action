We can first start by a simple triangle shape from which we've removed the bottom.
We can rotate that shape along the center point of the canvas.
Then at each 45 degrees, we can duplicate that shape.
We can exaggerate the multiplication, so it happens every 9 degrees, creating a complex star.
Taking the initial shape, we can add lines inside the shape.
We can repeat the same 90 degrees multiplication as earlier but with the more complex shape.
The more we duplicate the shape, the more details we're starting to see inside the shape.
With generous duplication, very interesting patterns start to appear out of the initial shape.
